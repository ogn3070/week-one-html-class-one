My Study Roadmap

  • Phase 1: Frontend Foundations (Weeks 1–4)
    1. Week 1: HTML Basics
      • Topics: structure, semantic tags, forms, links, images
      • Project: Simple personal webpage
      • Learning outcomes:
        • Write valid, semantic HTML pages
        • Know when to use headings, paragraphs, lists and forms
        • Make accessible images and links
    2. Week 2: CSS Basics
      • Topics: selectors, box model, typography, colors
      • Project: Styled personal webpage
      • Learning outcomes:
        • Apply CSS to change layout & visual style
        • Understand margin, padding, border and sizing
        • Use fonts and spacing for readable UI
    3. Week 3: Layout with Flexbox & Grid
      • Topics: flexbox, CSS Grid, responsive basics
      • Project: Responsive portfolio page
      • Learning outcomes:
        • Create responsive column/row layouts
        • Choose between flex and grid for common patterns
        • Use media queries for mobile-friendly pages
    4. Week 4: JavaScript Basics
      • Topics: variables, functions, DOM manipulation, events
      • Project: Interactive to-do list
      • Learning outcomes:
        • Read and update page content with the DOM
        • Handle user events (clicks, input)
        • Write simple JS functions to solve UI problems
  • Phase 2: Advanced Frontend (Weeks 5–6)
    1. Week 5: JavaScript ES6 & APIs
      • Topics: ES6 syntax, promises/async, fetch API
      • Project: Weather app using a public API
      • Learning outcomes:
        • Use modern JS features (let/const, arrow functions)
        • Fetch and display data from APIs
        • Handle asynchronous code correctly
    2. Week 6: React Basics
      • Topics: components, props, state, JSX
      • Project: React calculator or small CRUD UI
      • Learning outcomes:
        • Create reusable React components
        • Manage component state and props
        • Structure a small React app
  • Phase 3: Backend (Weeks 7–9)
    1. Week 7: Python & Django (intro)
      • Topics: Python basics, virtualenv, Django project setup
      • Project: Minimal Django app (hello world / simple views)
      • Learning outcomes:
        • Use Python for basic scripting and logic
        • Create and run a Django project
        • Understand request/response cycle on server
    2. Week 8: REST API Basics
      • Topics: routes, serializers (or JSON handling), HTTP methods
      • Project: Blog API (create/read endpoints)
      • Learning outcomes:
        • Design REST endpoints (GET, POST, PUT, DELETE)
        • Return and accept JSON data
        • Test APIs with Postman or curl
    3. Week 9: Database Integration
      • Topics: MySQL basics, models, migrations, CRUD
      • Project: Blog API with persistent storage
      • Learning outcomes:
        • Design simple relational schema
        • Perform CRUD operations from the backend
        • Connect Django to MySQL and run migrations
  • Phase 4: Full-Stack Integration (Weeks 10–12)
    1. Week 10: Connect Frontend & Backend
      • Topics: AJAX/Fetch from React to API, CORS basics
      • Project: Full-stack blog app (frontend + API)
      • Learning outcomes:
        • Make API calls from React to your backend
        • Handle data flow and errors across stack
        • Understand CORS and secure basic endpoints
    2. Week 11: Authentication & Accounts
      • Topics: user models, sessions, JWT, protected routes
      • Project: Add login/register to blog app
      • Learning outcomes:
        • Implement user authentication (sessions or JWT)
        • Protect routes and store tokens safely
        • Understand password hashing and simple security basics
    3. Week 12: Deployment & Portfolio
      • Topics: GitHub, CI basics, host on Heroku/Netlify/Vercel
      • Project: Deploy and showcase your final app
      • Learning outcomes:
        • Deploy a full-stack app to a public URL
        • Set up a GitHub repo with README and live link
        • Understand basic environment/config handling
